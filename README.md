# isola
[![Gem Version](https://badge.fury.io/rb/isola.svg)](https://badge.fury.io/rb/isola)

Super simple static site generator using erb.

## Installation

```bash
bundle add isola
```

If bundler is not being used to manage dependencies, install the gem by executing:

```bash
gem install isola
```

## Usage

1. place markdown files in your directory
2. `isola build`

See [Usage Guide](USAGE.md) / [使い方ガイド](USAGE.ja.md) for details.
