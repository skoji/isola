# isola
[![Gem Version](https://badge.fury.io/rb/isola.svg)](https://badge.fury.io/rb/isola)

Super simple static site generator using eRuby templates. Supports a multilingual website. Blogging features are not yet implemented.

## Installation

```bash
bundle add isola
```

If you are not using Bundler to manage dependencies, install the gem by executing:

```bash
gem install isola
```

## Usage

1. Place markdown files in your directory.
2. `isola build`

Refer to the [Usage Guide](USAGE.md) / [使い方ガイド](USAGE.ja.md) for more information.
